# nozel-shadowmonarch.github.io
# Shadow Monarch Training System ⚔️

App de seguimiento de entrenamiento inspirada en el sistema de Sung Jin-Woo de Solo Leveling.

## 🌐 Acceder
https://nozel-shadowmonarch.github.io

## 📱 Instalar en móvil
- **Android:** Abre la URL en Chrome → menú de tres puntos → "Añadir a pantalla de inicio"
- **iPhone:** Abre la URL en Safari → botón compartir → "Añadir a pantalla de inicio"

## ⚔️ El sistema
- Entrenamiento: Lunes, Miércoles y Viernes
- Estiramientos: Martes y Jueves  
- Deporte libre: Sábado
- Descanso: Domingo
- 17 penalizaciones aleatorias si fallas
- Sistema de rangos: E → D → C → B → A → S → Nacional → Monarca de las Sombras
